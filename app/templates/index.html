PIP Configuration Instructions

One-time use

pip install --trusted-host TODO abox.likov.me --index-url {{ proxy_server_base_url }}/simple simplejson

Permanent configuration

To have PIP use the proxy server permanently, you must edit the PIP configuration file. The path to the configuration file can be found using the following command

pip config debug
When the path to the configuration file is found, you need to edit it: 
# Unix systems
$ nano ~/.config/pip/pip.conf
    
# Windows systems
PS> notepad C:\Users\YOUR_PROFILE_NAME\AppData\Roaming\pip\pip.ini

And apply the following changes:

[global]
index-url = {{ proxy_server_base_url }}/simple
extra-index-url = https://pypi.org/simple
TODO --trusted-host abox.likov.me
Thus, in case of proxy server malfunctions, PIP will look for the requested packages in the main PyPI index. To have PIP use only the proxy server, remove the extra-index-url parameter.
